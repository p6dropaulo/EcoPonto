Project Summary EcoPonto is a donation management system designed to connect donors of recyclable materials with recycling associations, facilitating the collection and reuse of these materials. The project aims to enhance the donation process through a user-friendly API, enabling donors to register items and associations to collect them efficiently, thus promoting environmental sustainability.

Project Module Description Controllers: Handle incoming API requests and responses.

DoadorController: Manages donor-related operations. ItemController: Manages item-related operations, including linking items to donors. AssociacaoController: Manages associations and their ability to collect items. Services: Contain the business logic.

DoadorService: Handles operations related to donors. ItemService: Manages item operations, including associating items with donors. AssociacaoService: Handles operations related to associations and item collection. Repositories: Interface with the database.

DoadorRepository: CRUD operations for donors. ItemRepository: CRUD operations for items. AssociacaoRepository: CRUD operations for associations. DTOs: Data Transfer Objects for input and output data.

ItemCreateDto, ItemDto, DoadorCreateDto, DoadorDto, AssociacaoCreateDto, AssociacaoDto. Exception Handling: Global exception handler for consistent error responses.
